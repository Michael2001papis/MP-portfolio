## 📋 אודות הפרויקט

פורטפוליו אישי ומקצועי של **Michael Papismedov (MP)** — מפתח Frontend המתמחה באפליקציות ווב אינטראקטיביות, עיצוב רספונסיבי וממשקי משתמש. האתר מציג מגוון פרויקטים מעשיים שנבנו מ־HTML, CSS ו־JavaScript טהור.

## 🛠️ טכנולוגיות

- **HTML5** — סמנטיקה, נגישות
- **CSS3** — Flexbox, Grid, משתנים, מדיה קווריז, Reset יחיד
- **JavaScript (Vanilla)** — דף הבית ללא ספריות חיצוניות
- **Canvas API** — משחקים (Snake, Breakout, Pong)
- **ES Modules** — Addify, Countries
- **FormSubmit.co** — שליחת טפסים לאימייל
