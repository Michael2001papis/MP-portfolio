פורטפוליו אישי של מיכאל פפיסמדוב – אתר תדמית המציג פרויקטים ב-HTML ו-JavaScript.

העתקת קוד, עיצוב או תוכן אסורה. ראה [Terms](https://github.com/Michael2001papis/MP-portfolio/blob/main/legal.html) לפרטים.

## 🚀 אודות

אתר פורטפוליו הכולל:
- **כרטיס ביקור** – היכרות אישית
- **היכולות שלי** – HTML, CSS, JavaScript, SASS, Bootstrap
- **פרויקטי HTML** – 6 פרויקטים (רפואה, מיקרוסופט, אפל, צה"ל, חלל, AI)
- **פרויקטי JavaScript** – 9 פרויקטים (Countries, Memory Game, Calculator, Snake, ועוד)
- **צור קשר** – טופס יצירת קשר

## 📁 מבנה הפרויקט

```
MP-portfolio/
├── index.html              # דף הבית
├── First landing page/     # CSS ו-JS ראשי
├── pages/
│   ├── LandingPage/        # דפי פרויקטי HTML
│   ├── LandingPageJS/      # דפי פרויקטי JS
│   ├── JS project pages/   # הפרויקטים עצמם
│   └── עמודמעבריםכללי/    # ניווט בין פרויקטים
├── styles/                 # קבצי עיצוב
├── icon/                   # אייקונים
└── images/                 # תמונות ווידאו
```

## 🛠️ טכנולוגיות

- Bootstrap

## 📦 הרצה מקומית

1. שכפל את הפרויקט:
   ```bash
   git clone https://github.com/Michael2001papis/MP-portfolio.git
   cd MP-portfolio
   ```

2. פתח את `index.html` בדפדפן, או השתמש ב-Live Server (פורט 5502)
