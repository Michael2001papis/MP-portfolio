1. סקירה כללית
═══════════════════════════════════════════════════════════════════════════════

אתר פורטפוליו אישי של מיכאל פפיסמדוב - מפתח אתרים.
האתר מציג כרטיס ביקור, כישורים טכניים, פרויקטי HTML (6) ופרויקטי JavaScript (9).
האתר תומך בעברית (RTL) ובאנגלית, מותאם למובייל ומכיל פונקציית חיפוש.

מקור: https://github.com/Michael2001papis/MP-portfolio

═══════════════════════════════════════════════════════════════════════════════

   - צור קשר (#CU)
   - שדה חיפוש + כפתור "חפש"

3.2 כרטיס ביקור (#main)
   - שם: Michael Papismedov
   - טקסט היכרות (עברית)
   - כפתורים: פרויקטים HTML | פרויקטים JS | צור קשר

   - HTML 85%

   מוסתרים - כפתור "הצג עוד פרויקטים" (3): QP7-QP9
   QP7: Snakegame Nokia 6110 (וידאו רקע)
   QP8: Table Tennis (וידאו רקע)
   QP9: Calculator

3.6 צור קשר (#CU)
   - טופס: שם מלא, אימייל, טלפון, כפתור שלח

3.7 תפריט נייד (mobile)
   - מוצג במסכים < 750px
   - אייקונים: בית, כרטיס, יכולות, פרויקטים, JS, צור קשר

3.8 Footer
   - אתרי מידע: HTML, CSS, JS, Bootstrap, SASS (W3Schools)
   - מכווין עמוד: ניווט פנימי

4. פונקציונליות JavaScript (Main Menu.js)
═══════════════════════════════════════════════════════════════════════════════

4.1 חיפוש (input event)
   - חיפוש לפי הקלדה בשדה
   - מעבר לסקשנים: תפריט, היכולות שלי, HTML, JS, צור קשר
   - מעבר לעמודי פרויקטים: לפי שם הפרויקט

4.2 כפתור "הצג עוד פרויקטים"
   - לחיצה מציגה/מסתירה 3 פרויקטים נוספים (QP7, QP8, QP9)
   - טקסט מתחלף: "הצג עוד פרויקטים" / "הסתר פרויקטים נוספים"

הערה: כפתור "חפש" ב-HTML קורא ל-searchFunction() שלא מוגדר ב-JS.
      החיפוש עובד רק בהקלדה (addEventListener על input).

10. בעיות וחסרים שזוהו
═══════════════════════════════════════════════════════════════════════════════

1. תיקיות icon/ ו-images/ - יש לוודא שהן קיימות ומכילות את כל הקבצים.
2. searchFunction() - כפתור "חפש" קורא לפונקציה שלא קיימת ב-Main Menu.js.
3. שגיאת כתיב: ServicדCountries.JS (ד במקום e).
4. עקביות נתיבים: שימוש מעורב ב-/ ו-../ בקבצי JS.
5. טופס צור קשר - action="" (ריק), אין שליחה לשרת.
